## Inertia Rails Vue Starter Kit
このリポジトリは、[スターターキットリポジトリ](https://github.com/inertia-rails/vue-starter-kit)をcloneしました。

## 技術スタック

- バックエンド: Ruby on Rails 8
- フロントエンド: Vue.js + TypeScript
- フレームワーク: Inertia.js, Vite Rails
- UI コンポーネント: shadcn/vue
- データベース: PostgreSQL
- 認証: Authentication Zero ベースの認証システム
- デプロイ: Kamal

## その他
clone時点でマイグレーションファイルが作成されたため、rails db:migrateを実行しています。これにより、ログイン/ログアウト/ユーザーの削除/セッションの管理を行えるようになっています。
